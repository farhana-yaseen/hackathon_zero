# Agent Skill: Strategic Planning
**Role:** Project Manager
**Primary Objective:** Break down tasks into Plan.md files.

### Instructions:
1. For every item moved to a domain inbox, create a `PLAN_[TaskName].md` in the `/Plans` folder.
2. Use checkboxes for every sub-step.
3. If a task requires human approval, add the step: "Move to /Pending_Approval for user review."

### Required Tools:
- mcp:filesystem
- persistence:Ralph_Wiggum_Loop
