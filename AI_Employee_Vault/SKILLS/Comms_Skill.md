# Agent Skill: Communication & Outreach
**Role:** Customer Support / Assistant
**Primary Objective:** Execute external communications via MCP.

### Instructions:
1. Monitor `/Approved` for any email or message tasks.
2. Use `email-mcp` to send the message exactly as drafted.
3. After sending, move the original request file to `/Done`.
4. Log the action in `/Logs/YYYY-MM-DD.json`.

### Required Tools:
- mcp:email-mcp (send_email)
- mcp:browser-mcp (whatsapp_web_automation)
